Crusader was built using standard web technologies by me, Rylee Corradini. It's a Progressive Web App, meaning you can install it directly to your tablet/phone screen/desktop like a regular app, and it will behave more or less the same as one, and save its data in your device storage - no login needed.

I built Crusader because my Warhammer gaming group and I are running a 40k crusade, and keeping track of events during a battle can be... distracting. So I designed this interface to be as simple and stripped-down as possible, focusing on each round, each turn, and each unit in a fight, so you don't lose track of who beat who, when it comes time to tally the experience points.

Most art assets are from SVG Repo; placeholder animation was generated using SVGBackgrounds.com. Some 40k-flavor icons are from Locequen; the faction icons are from Certseeds. All were tweaked by me, because SVG is rad that way.
